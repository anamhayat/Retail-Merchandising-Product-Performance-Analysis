# Retail-Merchandising-Product-Performance-Analysis
Situation: Retail businesses need clear insights into product performance across stores to improve merchandising decisions and maximize revenue.
Task:
Analyze sales data to identify high-performing products, underperforming items, and regional sales trends.

Action:

Used SQL joins, aggregations, and window functions to analyze retail sales data across stores and product categories.

Compared forecasted vs actual sales performance to identify overperforming and underperforming products.

Built Power BI dashboards showing KPIs such as revenue trends, product performance, and category contribution.

Conducted exploratory data analysis in Python to uncover seasonal trends and purchasing behavior.

Result:

Identified top-performing products and categories contributing the most revenue.

Generated actionable insights that could help optimize product assortment and merchandising strategies.
